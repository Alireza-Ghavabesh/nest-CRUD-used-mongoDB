## Description

Simple CRUD Api base on (MongoDB and NestJs) <br />
dummy entities: musics and items


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
